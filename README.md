# Nmap Web Scanner

Nmap을 사용한 네트워크 스캔 기능을 웹 UI로 제공하는 도구입니다. 실시간 스캔 진행 상황 확인, 호스트 상세 정보(아코디언 뷰), 스캔 히스토리 관리 및 CSV 내보내기 기능을 지원합니다.

## ✨ 주요 기능

- **실시간 스캔**: Fetch API와 Stream을 사용하여 스캔 진행 상황을 실시간으로 출력합니다.
- **호스트 상세 조회**: 클릭 시 해당 호스트의 포트 및 서비스 버전 정보를 아코디언 형태로 보여줍니다.
- **스캔 히스토리**: 과거 스캔 이력을 보관하고 언제든지 다시 조회하거나 삭제할 수 있습니다.
- **CSV 내보내기**: 전체 스캔 결과를 CSV 파일로 다운로드하여 분석할 수 있습니다.
- **호스트 이름 탐지**: DNS 역방향 조회를 통해 네트워크 장치의 이름을 탐지합니다.

## 🛠️ 필수 설치 요소

1. **Python 3.8+**
2. **Nmap**: [nmap.org](https://nmap.org/download.html)에서 설치해야 합니다. 스캔 엔진의 핵심 도구입니다.

## 🚀 시작하기

### 1. 가상 환경 설정 및 패키지 설치

```bash
# 가상 환경 생성 (추천)
python -m venv .venv

# 가상 환경 활성화 (Windows)
.venv\Scripts\activate

# 의존성 패키지 설치
pip install -r requirements.txt
```

### 2. 애플리케이션 실행

**중요**: 윈도우에서 상세한 호스트 탐지를 위해서는 터미널을 **관리자 권한**으로 실행해야 합니다.

```bash
python app.py
```

브라우저에서 `http://127.0.0.1:5000`에 접속하여 사용하세요.

## 🔒 보안 주의 사항

- **인증 부재**: 이 애플리케이션에는 별도의 로그인이 없습니다. 외부 인터넷에 포트를 그대로 개방할 경우, 누구나 본인의 네트워크를 스캔할 수 있게 되므로 **반드시 로컬 환경에서만 사용**하거나 외부 노출 시 보안 프록시(Basic Auth 등)를 설정하세요.
- **입력 검증**: 기본적인 입력 값 검증이 추가되어 있지만, Nmap은 강력한 도구이므로 항상 허가된 대상에 대해서만 스캔을 수행해야 합니다.

## ⚠️ 법적 면책 조항

- 이 도구는 **허가된 네트워크**에서만 사용해야 합니다. 승인되지 않은 네트워크 스캔은 불법일 수 있습니다.
- 윈도우 환경에서 Nmap이 기본 경로(`C:\Program Files (x86)\Nmap`)에 설치되어 있지 않다면, 환경 변수(PATH)에 추가해야 할 수도 있습니다.
